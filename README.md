## Overview

This repository contains **Week 5 Assignment** at the **Celebal Technologies Data Engineering Internship**.
The assignment focuses on understanding **Apache Spark fundamentals** and performing common **data engineering operations** using **PySpark DataFrames**, including data cleaning, filtering, aggregation, schema modification, and transformation.

---

## Assignment Objectives

- Understand Spark fundamentals
- Compare Spark with traditional MapReduce
- Work with immutable Spark DataFrames
- Perform data cleaning operations
- Handle duplicate and null values
- Apply filtering techniques
- Use aggregation functions
- Perform GroupBy operations
- Modify DataFrame schemas
- Understand Shuffle and Wide Transformations
- Build an end-to-end data processing pipeline
